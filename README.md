# 100 MINEROS DIJERON — EDICIÓN GRUPO BACIS ⛏️🏆

¡Bienvenido a la aplicación oficial estilo **100 Mexicanos Dijeron** adaptada para la gran familia minera de **Grupo Bacis (Mina de Bacis, Durango)**!

## ✨ Características Principales
1. **Estética y Colores Grupo Bacis**:
   - Paleta oficial en azul rey profundo, plata y oro brillante con luces LED estilo estudio de televisión.
   - Emblema vectorial de **Grupo Bacis** (pala y pico cruzados en arco minero).
2. **Audios Oficiales Integrados**:
   - `a-jugar-100-mexicanos-dijeron_uzh3r4B.mp3` (Música de entrada / ¡A Jugar!)
   - `correcto-100-mexicanos-dijeron.mp3` (Efecto al girar tarjeta correcta)
   - `incorrecto-100-mexicanos-dijeron.mp3` (Efecto dramático al presionar Tache ❌)
   - `boton-10-mexicanos-digieron.mp3` (Efecto de botones y navegación)
   - `triunfo-100-mexicanos-dijeron.mp3` (Victoria / Sumar ronda)
3. **Más de 20 Preguntas Mineras Auténticas** categorizadas en:
   - 🗣️ **Modismos & Vida Cotidiana** (el lonche, el chivo, apodos mineros, salidas de turno)
   - 🕳️ **Mina Subterránea** (Scooptram, Jumbo, ademe, acicalado, frentes)
   - ⚙️ **Planta de Beneficio** (molienda, flotación, ley de oro/plata, reactivos)
   - 💎 **Geología & Exploración** (vetas, bonanza, pirita, martillo de geólogo)
   - 🦺 **Seguridad & EPP** (casco, autorrescatador, reglas de oro)
4. **Tablero 3D Interactivo**:
   - Tarjetas volteables con animación 3D.
   - Marcador para 2 equipos (**LOS BARRETEROS** vs **LOS GAMBUSINOS**, con nombres editables al hacer clic sobre ellos).
   - Contador de puntos acumulados en la ronda y botones para sumarlos al equipo ganador.
   - Panel de 3 Taches (❌ ❌ ❌) con animación dramática a pantalla completa.
5. **Creador de Preguntas Personalizadas**:
   - Botón **"➕ CREAR PREGUNTA"** en la cabecera que permite agregar nuevas encuestas y guardarlas automáticamente en el navegador (`localStorage`).

## 🚀 Cómo Subir directamente a Vercel

Esta aplicación está lista para producción (`Zero-Config`) para subir a **Vercel**:

### Opción A: Con la CLI de Vercel (Recomendado y más rápido)
Si tienes Node.js instalado, abre tu terminal en esta carpeta y ejecuta:
```bash
npx vercel
```
1. Presiona **Enter** para confirmar la configuración por defecto.
2. ¡Listo! Vercel te dará una URL pública en segundos.

### Opción B: Subiendo a GitHub + Vercel
1. Sube esta carpeta a un repositorio en tu **GitHub**.
2. En [vercel.com](https://vercel.com), selecciona **Add New Project** e importa tu repositorio.
3. Haz clic en **Deploy** (no requiere ningún comando de compilación, se detecta automáticamente como sitio estático optimizado).

## 🎮 Guía Rápida para el Conductor del Juego
- **Renombrar Equipos**: Haz clic en "LOS BARRETEROS" o "LOS GAMBUSINOS" para cambiarles el nombre.
- **Revelar Respuesta**: Haz clic en el número de la tarjeta azul para voltearla, sumar los puntos y escuchar el sonido de acierto.
- **Marcar Error (Strike)**: Presiona el botón rojo **"❌ TACHE / ERROR"** para mostrar la X gigante y reproducir la alarma de error.
- **Sumar Ronda al Equipo**: Al terminar una pregunta, presiona **"🏆 SUMAR RONDA"** en el equipo ganador para acreditarle los puntos en juego.
